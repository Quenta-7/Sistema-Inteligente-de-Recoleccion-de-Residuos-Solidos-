├─ IMPLEMENTACIÓN (4 SPRINTS = 6 SEMANAS)
│  ├─ Sprint 1: Auth + BD
│  ├─ Sprint 2: Zonas/Horarios
│  ├─ Sprint 3: Reportes
│  └─ Sprint 4: Admin + Pulido
│
├─ ENTREGABLES
│  ├─ Semana 6: MVP 50% + Doc Cap I-II
│  ├─ Semana 12: MVP 100% + Doc Cap I-III
│  └─ Semana 16: FINAL + Doc completa + Exposición

   ├─ Social: Empodera ciudadanía
